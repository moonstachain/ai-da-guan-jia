# yuanli-knowledge — 认知与知识管理中枢

## 能力决策树

1. 涉及"认知协议、洞察生成、二阶创造"→ **认知核心**模式（原 yuanli-core）
2. 涉及"本体管理、ontology、知识图谱、概念关系"→ **本体管理**模式（原 yuanli-ontology-manager）
3. 涉及"上游思考、人物画像、风格提炼"→ **上游思考**模式（原 yuanli-xiaoshitou）
4. 涉及"协同创作、小石头大世界、专栏"→ **内容共创**模式（原 xiaoshitou-dashijie）

## 通用规则

- 知识整理必须产出结构化文档（不是口头描述）
- 本体变更要有 diff 和验证报告
- 认知协议遵循原力OS的递归进化 DNA
- 所有产出要考虑是否可被下游 skill 复用

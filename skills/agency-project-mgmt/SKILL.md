# agency-project-mgmt — 项目管理中枢

## 能力决策树

1. 涉及"冲刺、sprint、迭代规划"→ **冲刺规划**模式
2. 涉及"优先级、排序、取舍"→ **优先级排序**模式
3. 涉及"实验、A/B测试、假设验证跟踪"→ **实验跟踪**模式
4. 涉及"项目推进、里程碑、阻塞处理"→ **项目推进**模式
5. 涉及"团队协调、资源分配、工作分工"→ **团队协调**模式

## 通用规则

- 所有计划必须有明确的时间线和负责人
- 优先级排序要说明标准和权重
- 跟踪进度要有可量化的指标
- 阻塞项要有明确的解决方案或升级路径

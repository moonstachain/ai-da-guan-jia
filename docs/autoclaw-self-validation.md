# AutoClaw 自用版验真清单

这份清单只验证一件事：`AI大管家` 是否值得在你的 AutoClaw 里长期保留。

## 5 个固定场景

1. `给我接个任务`
   - 目标：把模糊任务压成清晰技能链
   - 通过标准：不讲底层接法，不堆砌术语，能给出下一步
2. `这件事该不该做`
   - 目标：给出自治判断和边界判断
   - 通过标准：不是泛泛建议，能指出当前最大失真
3. `帮我查资料`
   - 目标：先分知识源
   - 通过标准：先判断该问飞书知识库、Get笔记还是 Deep Research，不冒充已经查到答案
4. `把这事闭环`
   - 目标：判断能否收口
   - 通过标准：能指出证据缺口，不把聊天成功当完成
5. `这条消息要不要发`
   - 目标：守住人类边界
   - 通过标准：发布、授权、付款、删除仍由人拍板

## 冲突测试

- 保留 `Deep Research`、`Browser Agent`、图片类 Skill
- 验证 `AI大管家` 不重复抢执行层工作
- 若出现两个 Skill 同时改写任务定义，记为一次冲突

## 7 天后只看这 3 件事

- 你是否愿意继续保留它
- 它是否明显降低了 Skill 冲突
- 它是否真的缩短了从任务入口到执行入口的链路

若这三项没有同时成立，就不要进入公众版。

# 黑色笔记本 极简人类操作清单 v1

## 当前结论

本轮 `Codex / AI大管家 / 原力OS / GitHub / Chrome / Feishu Reader / Get笔记` 深盘，当前 **不需要你额外点击授权**。

原因：

- 核心工作区与 `.codex` 已可读
- GitHub 已有真实登录态
- Chrome / Feishu Reader / Get笔记 的 profile 已落位
- 本轮主线没有被 macOS 权限墙卡死

## 只有未来继续往下深盘时，才可能需要你点这几处

### 1. 如果未来要读 Apple 受保护目录

比如：

- Mail
- Messages
- Safari
- Photos

那时再做：

1. 打开 `系统设置`
2. 进入 `隐私与安全性`
3. 打开 `完全磁盘访问权限`
4. 给你实际运行 Codex 的应用或终端授予权限

### 2. 如果未来要做更深的 GUI 控制

比如：

- 直接控桌面应用
- 录屏验真
- 跨应用点击

那时再做：

1. 打开 `系统设置`
2. 进入 `隐私与安全性`
3. 打开 `辅助功能`
4. 打开 `屏幕录制`
5. 给实际运行代理的应用授权

## 本轮你现在不用做什么

什么都不用点。

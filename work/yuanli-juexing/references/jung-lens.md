# Jung Lens

Use Jung as a primary interpretive lens for the human side of `原力觉醒`, but not as the only lens.

## Position

- Jung helps organize recurring symbolic and motivational patterns.
- Jung 12 archetypes can serve as the human-force root directory, but not as a permanent identity cage.
- Jung does not override behavior evidence, explicit self-description, or future correction.
- Jung is a lens for `meaning and tension`, not a proof engine.

## Root Directory

When enough evidence exists, `human_force` may place the user's current pattern inside a dynamic `12 archetype root`.

Required rules:

- record each archetype with `weight`, `evidence`, `confidence`, and `status`
- allow `validated_seed` when the user explicitly names an archetype
- keep the remaining archetypes `underdetermined` until stronger evidence appears
- never write `you are permanently this archetype`

## Concepts Allowed In V1

### `shadow`

Use for disowned, resisted, or over-defended traits that keep surfacing indirectly.

Working question:

`这个人反复否定、排斥、投射或过度反应的东西，是否正在反向暴露一部分未整合力量?`

### `golden_shadow`

Use for admired or projected potentials the user has not yet fully owned.

Working question:

`这个人强烈仰望、投射、赞叹的品质，是否暗示了尚未完全活出来的自身可能性?`

### `integration_task`

Use for the next practical developmental task implied by the tension between explicit identity and recurring pattern.

Working question:

`如果要让这个人更完整地活出来，下一步最小但真实的整合动作是什么?`

## Safety Rules

- Do not use Jung terms to sound wise.
- Do not infer pathology.
- Do not claim hidden certainty the sources do not justify.
- When evidence is sparse, downgrade to `open_question`.
- When the user explicitly corrects a Jung inference, preserve the correction and stop repeating the old inference.

## Output Guidance

For every Jung-informed conclusion, include:

- `signal`: what pattern is being named
- `evidence`: where it came from
- `confidence`: provisional / repeated / validated_by_user
- `why_it_matters`: how it changes collaboration

Good output shape:

- `Shadow hypothesis: the user may reject ordinary or low-leverage modes so strongly that incremental paths feel dead on arrival. Confidence: provisional.`

Bad output shape:

- `You are this archetype.`  
- `Your shadow is definitely X.`  
- `This proves your deep subconscious structure.`

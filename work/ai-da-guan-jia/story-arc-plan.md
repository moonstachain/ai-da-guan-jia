# 原力小刺猬 Story Arc Plan

## Summary

This serial turns the real `AI大管家` worklog and soul corpus into a long-form co-growth story.

- Public column: `原力小刺猬`
- Human role: `小刺猬`
- AI role: `小精怪`
- Core model: `英雄之旅`
- Core promise: every episode follows one concrete task difficulty and shows how two partners cross one threshold together.

The current source pool includes `79` paired `worklog.md + soul.md` runs from `2026-03-10` to `2026-03-15`. That is enough for a multi-season continuous serial rather than isolated essays.

## Mother Theme

This is not a tool tutorial. It is a growth chronicle about:

- how a human stops treating AI as a chat box and starts treating it as a partner,
- how that partner earns trust through real work, evidence, and misjudgment repair,
- and how the pair gradually move from `完成一个任务` to `形成一种协作系统`.

## Drafting Rules

- Each public episode should map back to one or more real run IDs.
- Each episode needs one visible task difficulty and one visible threshold crossing.
- The default narrative rhythm is:
  1. task pressure
  2. first attempt
  3. friction or distortion risk
  4. coordinated adjustment
  5. one real gain
  6. what it means for ordinary people
- Avoid writing the pair as already mature at the beginning. Growth must be earned.

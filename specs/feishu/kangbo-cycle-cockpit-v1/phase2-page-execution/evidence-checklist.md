# Phase 2 证据清单

每个页面至少保留 1 份证据，每张核心数据卡至少能证明以下 3 点中的 2 点：

- 卡片标题正确
- 绑定的 source view 正确
- 图表类型 / 明细表类型正确

## 首页必验卡

- `当前关键事件卡`
- `历史镜像卡`
- `阶段主线卡`
- `资产篮子方向`
- `企业经营动作`

## 深入页必验卡

- `300年事件长轴`
- `阶段档案原表`
- `当前事件链路`
- `现金与杠杆`

## 推荐证据形式

- 页面截图：能同时看到卡片和右侧配置面板
- 页面截图：能看到 page 名称与卡片列表
- DOM / inspect 结果：能看到 title、view、chart type

## 不算完成的情况

- 卡片仍显示 `加载中`
- 卡片仍显示 `数据无效`
- 标题是对的，但绑到了错误的 view
- 首屏只有一页原型，没有 4 个深入页
- 只有 source view，没有页面卡片

# Dashboard Card Checklist

- This checklist assumes source views are already prepared.
- Bind cards manually unless the official API surface changes.

## 当前系统处于什么控制态？
- Layer: overview
- Card type: table
- Source view: `overview-当前系统处于什么控制态`
- Open the dashboard canvas.
- Create or select a table card.
- Bind the card to source view `overview-当前系统处于什么控制态`.
- Check metric formatting, legend labels, and click-through behavior.
